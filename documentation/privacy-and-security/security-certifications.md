# Overview

Mailtrap maintains industry-leading security certifications, demonstrating our commitment to the highest standards of information security and data protection.

For detailed information about our security certifications, compliance documents, and security practices, visit our Trust Portal.

<a href="https://trust.mailtrap.io/" class="button primary">Visit Trust Portal</a>

## ISO/IEC 27001:2022

Mailtrap is ISO/IEC 27001:2022 certified, meeting all requirements for information security management.

## SOC 2 Type II

Mailtrap is currently in the process of obtaining SOC 2 Type II certification, with an estimated completion by the end of Q1 2026. The certification status will be updated in our Trust Portal.
