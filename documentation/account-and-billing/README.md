## Common Billing Tasks

### Update Payment Method
1. Go to Billing Settings
2. Click "Payment Methods"
3. Add new card or remove old
4. Set as default
5. Verify with test charge

### Change Billing Information
1. Navigate to Account Settings
2. Select "Billing Information"
3. Update company details
4. Save changes
5. Verify on next invoice

### View Usage Statistics
1. Open Dashboard
2. Click "Usage"
3. Select date range
4. View detailed breakdown
5. Export if needed
