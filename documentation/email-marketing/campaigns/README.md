## Performance Metrics

### Key Metrics to Track

* **Open Rate**: Email opens / Emails delivered
* **Click Rate**: Unique clicks / Emails delivered
* **Conversion Rate**: Conversions / Emails delivered
* **Bounce Rate**: Bounces / Emails sent
* **Unsubscribe Rate**: Unsubscribes / Emails delivered

### Benchmarks by Industry
