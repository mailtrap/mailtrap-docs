# OneLogin

## Overview

This guide walks you through setting up Single Sign-On (SSO) between Mailtrap and OneLogin using a SAML custom connector.

{% stepper %}
{% step %}

{% endstep %}

{% step %}

<div align="left" data-with-frame="true"><img src="../../../.gitbook/assets/setup-sso-with-onelogin-3.png" alt="" width="563"></div>
